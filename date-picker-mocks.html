Vanilla web component

Date picker with day, month, and year zoom

This page now uses a real <date-picker> element instead of static mocks. The component keeps its own state, supports keyboard navigation, and exposes value, min, max, locale, and first-day-of-week hooks.

Keyboard map

Interaction model

  • Arrow keys move across the active grid.
  • Enter selects a day or drills into the next zoom level.
  • Escape moves back from year -> month -> day, then closes.
  • Month label in day view zooms out; year label in month view zooms out again.
Component contract

Attributes and API

  • Attributes: value, min, max, locale, first-day-of-week, open
  • Methods: showPicker(), hidePicker(), togglePicker()
  • Events: input, change
